API

Une API REST a été mise en place pour ouvrir les données du projet FreeMedForms, elle se limite pour l'instant à une recherche par nom de médicaments, et la récuperation de son UID, elle s'utilise comme le montrent les exemples ci-après :

L'url générale est /api/drugs.{format_desiré}/{debut_du_nom_du_medicament} ou encore sous la forme suivante /api/drugs.{format_desiré}?limit={nombre_limite_de_resultats}&q={nom}, exemples :

  • /api/drugs.json/aspirine
  • /api/drugs.json?limit=10&q=aspirine
Pour des raisons de performance les requêtes sont limitées à 50 résultats maximum.
